# Phase 4: Full NIP-46 HSM — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the ESP32 a complete NIP-46 signing bunker with multi-master support, on-device NIP-44 transport encryption, client approval policies, and the full heartwood extension method set. The Pi becomes a zero-trust encrypted pipe.

**Architecture:** Multiple master secrets stored in NVS (up to 8, three provisioning modes). Per-request routing by pubkey — the bridge includes the target master in each encrypted frame. NIP-44 transport encryption happens on-device so the Pi never sees plaintext or keys. Client approval policies are pushed from the bridge at session start and held in RAM only. Three approval tiers: auto-approve, OLED-notify, and button-required.

**Tech Stack:** Rust, ESP-IDF v5.x (std), secp256k1 (C FFI), chacha20poly1305, hkdf, x25519-dalek, aes, cbc, base64, serde/serde_json, ssd1306/embedded-graphics.

**Spec:** `docs/specs/2026-04-03-hsm-design.md`

**Testing strategy:** Tasks 1–5 are host-testable in `common/` via `cargo test`. Tasks 6–11 are firmware that requires hardware verification. Tasks 12–13 update the bridge and test harness (host-testable). Task 14 is docs/commit.

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
#[repr(u8)]
pub enum MasterMode {
    /// Raw nsec stored as-is — vanilla NIP-46, no tree derivation.
    Bunker = 0,
    /// BIP-39 mnemonic → derived root secret.
    TreeMnemonic = 1,
    /// Existing nsec → HMAC → tree root.
    TreeNsec = 2,
}

impl MasterMode {
    pub fn from_u8(v: u8) -> Option<Self> {
        match v {
            0 => Some(Self::Bunker),
            1 => Some(Self::TreeMnemonic),
            2 => Some(Self::TreeNsec),
            _ => None,
        }
    }

    pub fn is_tree(&self) -> bool {
        matches!(self, Self::TreeMnemonic | Self::TreeNsec)
    }
}

// common/src/nip44.rs
//
// NIP-44 v2 encryption: conversation key derivation + XChaCha20-Poly1305 AEAD.
//
// secp256k1 secret keys are converted to X25519 scalars for ECDH.
// The shared point is fed through HKDF-SHA256 to produce a 32-byte
// conversation key, which is then used for XChaCha20-Poly1305.

use chacha20poly1305::aead::{Aead, KeyInit};
use chacha20poly1305::XChaCha20Poly1305;
use hkdf::Hkdf;
use sha2::Sha256;
use x25519_dalek::{PublicKey as X25519PublicKey, StaticSecret as X25519StaticSecret};

/// NIP-44 protocol version.
const NIP44_VERSION: u8 = 2;

/// HKDF salt for NIP-44 conversation key derivation.
const HKDF_SALT: &[u8] = b"nip44-v2";

/// Derive the X25519 static secret from a 32-byte secp256k1 private key.
///
/// X25519 clamping is applied internally by `x25519-dalek`.
fn to_x25519_secret(secp_secret: &[u8; 32]) -> X25519StaticSecret {
    X25519StaticSecret::from(*secp_secret)
}

/// Convert a 32-byte secp256k1 x-only public key to an X25519 public key.
///
/// This performs the secp256k1 point → Curve25519 Montgomery point conversion.
/// The x-only pubkey is first decompressed to a full secp256k1 point (assuming
/// even y-coordinate), then the x-coordinate is mapped to Montgomery form.
///
/// NOTE: This is a simplified approach. The full NIP-44 spec uses the `nostr`
/// crate's conversion which handles the field arithmetic correctly. We replicate
/// the same byte-level operation here.
fn to_x25519_pubkey(secp_xonly_pubkey: &[u8; 32]) -> X25519PublicKey {
    // For NIP-44, the conversation key derivation uses the secp256k1 private key
    // directly as an X25519 scalar (with clamping) and the peer's secp256k1 x-only
    // pubkey bytes directly as an X25519 point. This is NOT mathematically correct
    // curve conversion, but it matches what the nostr crate does: the private key
    // bytes are used as-is for X25519.
    //
    // IMPORTANT: The actual NIP-44 conversation key derivation computes a shared
    // point on secp256k1 first (ECDH), then extracts the x-coordinate, then runs
    // HKDF. It does NOT use X25519 at all. The x25519-dalek crate is not needed.
    //
    // We will implement the correct approach: secp256k1 ECDH → HKDF.
    X25519PublicKey::from(*secp_xonly_pubkey)
}

/// Derive the NIP-44 conversation key from our secret and peer's public key.
///
/// 1. secp256k1 ECDH → 32-byte shared x-coordinate
/// 2. HKDF-SHA256(salt="nip44-v2", ikm=shared_x) → 32-byte conversation key
pub fn get_conversation_key(
    our_secret: &[u8; 32],
    peer_pubkey: &[u8; 32],
) -> Result<[u8; 32], &'static str> {
    // Step 1: secp256k1 ECDH — compute shared point, extract x-coordinate.
    // We use the secp256k1 crate (already a dep via feature) for this.
    #[cfg(feature = "secp256k1-backend")]
    {
        use secp256k1::{Secp256k1, SecretKey, PublicKey};

        let secp = Secp256k1::new();
        let sk = SecretKey::from_slice(our_secret)
            .map_err(|_| "invalid secret key")?;

        // Decompress x-only pubkey to full pubkey (assume even y).
        let mut full_pk_bytes = [0u8; 33];
        full_pk_bytes[0] = 0x02; // even y prefix
        full_pk_bytes[1..].copy_from_slice(peer_pubkey);
        let pk = PublicKey::from_slice(&full_pk_bytes)
            .map_err(|_| "invalid peer pubkey")?;

        // ECDH: shared_point = sk * pk, extract x-coordinate.
        let shared_point = secp256k1::ecdh::shared_secret_point(&pk, &sk);
        // shared_secret_point returns 64 bytes (x || y), we want just x (first 32).
        let shared_x: [u8; 32] = shared_point[..32].try_into().unwrap();

        // Step 2: HKDF-SHA256
        let hk = Hkdf::<Sha256>::new(Some(HKDF_SALT), &shared_x);
        let mut conversation_key = [0u8; 32];
        hk.expand(b"", &mut conversation_key)
            .map_err(|_| "HKDF expand failed")?;

        Ok(conversation_key)
    }

    #[cfg(feature = "k256-backend")]
    {
        use k256::ecdh::diffie_hellman;
        use k256::elliptic_curve::sec1::FromEncodedPoint;
        use k256::{EncodedPoint, NonZeroScalar, PublicKey as K256PublicKey};

        // Build compressed point from x-only pubkey (assume even y).
        let mut compressed = [0u8; 33];
        compressed[0] = 0x02;
        compressed[1..].copy_from_slice(peer_pubkey);
        let encoded = EncodedPoint::from_bytes(&compressed)
            .map_err(|_| "invalid encoded point")?;
        let pk = K256PublicKey::from_encoded_point(&encoded);
        if pk.is_none().into() {
            return Err("invalid peer pubkey");
        }
        let pk = pk.unwrap();

        let sk = NonZeroScalar::try_from(our_secret.as_slice())
            .map_err(|_| "invalid secret key")?;
        let shared = diffie_hellman(&sk, pk.as_affine());
        let shared_bytes: &[u8] = shared.raw_secret_bytes().as_slice();

        let hk = Hkdf::<Sha256>::new(Some(HKDF_SALT), shared_bytes);
        let mut conversation_key = [0u8; 32];
        hk.expand(b"", &mut conversation_key)
            .map_err(|_| "HKDF expand failed")?;

        Ok(conversation_key)
    }
}

/// Encrypt a plaintext message using NIP-44 v2.
///
/// Returns the ciphertext as a base64-encoded string (version byte + nonce + ciphertext + tag).
pub fn encrypt(
    conversation_key: &[u8; 32],
    plaintext: &str,
    nonce: &[u8; 24],
) -> Result<String, &'static str> {
    // Pad the plaintext per NIP-44 spec.
    let padded = pad(plaintext.as_bytes());

    // Derive message keys from conversation key + nonce via HKDF.
    let (chacha_key, chacha_nonce, hmac_key) = derive_message_keys(conversation_key, nonce)?;

    // Encrypt with XChaCha20-Poly1305.
    let cipher = XChaCha20Poly1305::new_from_slice(&chacha_key)
        .map_err(|_| "cipher init failed")?;
    let chacha_nonce = chacha20poly1305::XNonce::from_slice(&chacha_nonce);
    let ciphertext = cipher
        .encrypt(chacha_nonce, padded.as_ref())
        .map_err(|_| "encryption failed")?;

    // Build output: version(1) || nonce(24) || ciphertext(N)
    let mut output = Vec::with_capacity(1 + 24 + ciphertext.len());
    output.push(NIP44_VERSION);
    output.extend_from_slice(nonce);
    output.extend_from_slice(&ciphertext);

    // HMAC-SHA256 over the output for the MAC.
    use hmac::{Hmac, Mac};
    let mut mac = Hmac::<Sha256>::new_from_slice(&hmac_key)
        .map_err(|_| "HMAC init failed")?;
    mac.update(&output);
    let tag = mac.finalize().into_bytes();
    output.extend_from_slice(&tag);

    // Base64 encode the whole thing.
    use base64::Engine;
    Ok(base64::engine::general_purpose::STANDARD.encode(&output))
}

/// Decrypt a NIP-44 v2 ciphertext.
///
/// Input is a base64-encoded string. Returns the plaintext.
pub fn decrypt(
    conversation_key: &[u8; 32],
    ciphertext_b64: &str,
) -> Result<String, &'static str> {
    use base64::Engine;
    let data = base64::engine::general_purpose::STANDARD
        .decode(ciphertext_b64)
        .map_err(|_| "invalid base64")?;

    if data.len() < 1 + 24 + 32 {
        return Err("ciphertext too short");
    }

    let version = data[0];
    if version != NIP44_VERSION {
        return Err("unsupported NIP-44 version");
    }

    // Split: version(1) || nonce(24) || encrypted(N) || hmac(32)
    let nonce: [u8; 24] = data[1..25].try_into().unwrap();
    let mac_start = data.len() - 32;
    let encrypted = &data[25..mac_start];
    let received_mac = &data[mac_start..];

    // Derive message keys.
    let (chacha_key, chacha_nonce, hmac_key) = derive_message_keys(conversation_key, &nonce)?;

    // Verify HMAC before decrypting.
    use hmac::{Hmac, Mac};
    let mut mac = Hmac::<Sha256>::new_from_slice(&hmac_key)
        .map_err(|_| "HMAC init failed")?;
    mac.update(&data[..mac_start]);
    mac.verify_slice(received_mac)
        .map_err(|_| "HMAC verification failed")?;

    // Decrypt with XChaCha20-Poly1305.
    let cipher = XChaCha20Poly1305::new_from_slice(&chacha_key)
        .map_err(|_| "cipher init failed")?;
    let chacha_nonce = chacha20poly1305::XNonce::from_slice(&chacha_nonce);
    let padded = cipher
        .decrypt(chacha_nonce, encrypted)
        .map_err(|_| "decryption failed")?;

    // Unpad.
    let plaintext = unpad(&padded)?;
    String::from_utf8(plaintext.to_vec())
        .map_err(|_| "decrypted content is not valid UTF-8")
}

/// Derive chacha_key (32), chacha_nonce (24), hmac_key (32) from conversation key + nonce.
fn derive_message_keys(
    conversation_key: &[u8; 32],
    nonce: &[u8; 24],
) -> Result<([u8; 32], [u8; 24], [u8; 32]), &'static str> {
    let hk = Hkdf::<Sha256>::new(Some(nonce), conversation_key);
    let mut keys = [0u8; 88]; // 32 + 24 + 32
    hk.expand(b"nip44-v2", &mut keys)
        .map_err(|_| "HKDF expand failed")?;

    let chacha_key: [u8; 32] = keys[..32].try_into().unwrap();
    let chacha_nonce: [u8; 24] = keys[32..56].try_into().unwrap();
    let hmac_key: [u8; 32] = keys[56..88].try_into().unwrap();

    Ok((chacha_key, chacha_nonce, hmac_key))
}

/// Pad plaintext per NIP-44 v2 spec.
/// Format: [u16_be_unpadded_len][plaintext][zero_padding_to_next_power_of_2]
fn pad(plaintext: &[u8]) -> Vec<u8> {
    let unpadded_len = plaintext.len();
    let padded_len = calc_padded_len(unpadded_len);
    let mut out = Vec::with_capacity(2 + padded_len);
    out.extend_from_slice(&(unpadded_len as u16).to_be_bytes());
    out.extend_from_slice(plaintext);
    out.resize(2 + padded_len, 0);
    out
}

/// Unpad plaintext per NIP-44 v2 spec.
fn unpad(padded: &[u8]) -> Result<&[u8], &'static str> {
    if padded.len() < 2 {
        return Err("padded data too short");
    }
    let unpadded_len = u16::from_be_bytes([padded[0], padded[1]]) as usize;
    if unpadded_len == 0 || 2 + unpadded_len > padded.len() {
        return Err("invalid padding length");
    }
    Ok(&padded[2..2 + unpadded_len])
}

/// Calculate the padded length per NIP-44: next power of 2, minimum 32.
fn calc_padded_len(unpadded_len: usize) -> usize {
    if unpadded_len <= 32 {
        return 32;
    }
    let next_power = (unpadded_len as u32).next_power_of_two() as usize;
    // NIP-44 uses a more nuanced padding scheme — quantise to reduce leakage.
    // Simplified: pad to next power of 2.
    next_power
}

// common/src/nip04.rs
//
// NIP-04 legacy encryption: AES-256-CBC + PKCS7 padding + base64.
//
// DEPRECATED — NIP-44 should be preferred. This exists only for
// compatibility with older Nostr clients.

use aes::Aes256;
use cbc::cipher::{BlockDecryptMut, BlockEncryptMut, KeyIvInit};
use sha2::{Digest, Sha256};

type Aes256CbcEnc = cbc::Encryptor<Aes256>;
type Aes256CbcDec = cbc::Decryptor<Aes256>;

/// Derive the NIP-04 shared secret from our secp256k1 secret and peer's x-only pubkey.
///
/// NIP-04 uses the raw x-coordinate of the ECDH shared point (no HKDF),
/// which is why it's insecure — but we must match the spec.
pub fn get_shared_secret(
    our_secret: &[u8; 32],
    peer_pubkey: &[u8; 32],
) -> Result<[u8; 32], &'static str> {
    #[cfg(feature = "secp256k1-backend")]
    {
        use secp256k1::{Secp256k1, SecretKey, PublicKey};

        let secp = Secp256k1::new();
        let sk = SecretKey::from_slice(our_secret)
            .map_err(|_| "invalid secret key")?;

        let mut full_pk_bytes = [0u8; 33];
        full_pk_bytes[0] = 0x02;
        full_pk_bytes[1..].copy_from_slice(peer_pubkey);
        let pk = PublicKey::from_slice(&full_pk_bytes)
            .map_err(|_| "invalid peer pubkey")?;

        let shared_point = secp256k1::ecdh::shared_secret_point(&pk, &sk);
        let shared_x: [u8; 32] = shared_point[..32].try_into().unwrap();
        Ok(shared_x)
    }

    #[cfg(feature = "k256-backend")]
    {
        use k256::ecdh::diffie_hellman;
        use k256::elliptic_curve::sec1::FromEncodedPoint;
        use k256::{EncodedPoint, NonZeroScalar, PublicKey as K256PublicKey};

        let mut compressed = [0u8; 33];
        compressed[0] = 0x02;
        compressed[1..].copy_from_slice(peer_pubkey);
        let encoded = EncodedPoint::from_bytes(&compressed)
            .map_err(|_| "invalid encoded point")?;
        let pk = K256PublicKey::from_encoded_point(&encoded);
        if pk.is_none().into() {
            return Err("invalid peer pubkey");
        }
        let pk = pk.unwrap();

        let sk = NonZeroScalar::try_from(our_secret.as_slice())
            .map_err(|_| "invalid secret key")?;
        let shared = diffie_hellman(&sk, pk.as_affine());
        let shared_bytes: [u8; 32] = shared.raw_secret_bytes().as_slice().try_into().unwrap();
        Ok(shared_bytes)
    }
}

/// Encrypt a plaintext message using NIP-04 (AES-256-CBC).
///
/// Returns `"<base64_ciphertext>?iv=<base64_iv>"`.
pub fn encrypt(
    shared_secret: &[u8; 32],
    plaintext: &str,
    iv: &[u8; 16],
) -> Result<String, &'static str> {
    use base64::Engine;

    let plaintext_bytes = plaintext.as_bytes();

    // PKCS7 pad to AES block size (16 bytes).
    let block_size = 16;
    let pad_len = block_size - (plaintext_bytes.len() % block_size);
    let mut padded = Vec::with_capacity(plaintext_bytes.len() + pad_len);
    padded.extend_from_slice(plaintext_bytes);
    padded.resize(plaintext_bytes.len() + pad_len, pad_len as u8);

    // Encrypt.
    let cipher = Aes256CbcEnc::new_from_slices(shared_secret, iv)
        .map_err(|_| "AES init failed")?;
    let ciphertext = cipher.encrypt_padded_vec_mut::<cbc::cipher::block_padding::NoPadding>(&padded);

    let ct_b64 = base64::engine::general_purpose::STANDARD.encode(&ciphertext);
    let iv_b64 = base64::engine::general_purpose::STANDARD.encode(iv);

    Ok(format!("{}?iv={}", ct_b64, iv_b64))
}

/// Decrypt a NIP-04 ciphertext.
///
/// Input format: `"<base64_ciphertext>?iv=<base64_iv>"`.
pub fn decrypt(
    shared_secret: &[u8; 32],
    ciphertext: &str,
) -> Result<String, &'static str> {
    use base64::Engine;

    let parts: Vec<&str> = ciphertext.split("?iv=").collect();
    if parts.len() != 2 {
        return Err("invalid NIP-04 format: expected 'ciphertext?iv=iv'");
    }

    let ct_bytes = base64::engine::general_purpose::STANDARD
        .decode(parts[0])
        .map_err(|_| "invalid base64 ciphertext")?;
    let iv_bytes = base64::engine::general_purpose::STANDARD
        .decode(parts[1])
        .map_err(|_| "invalid base64 IV")?;

    if iv_bytes.len() != 16 {
        return Err("IV must be 16 bytes");
    }
    let iv: [u8; 16] = iv_bytes.try_into().unwrap();

    // Decrypt.
    let cipher = Aes256CbcDec::new_from_slices(shared_secret, &iv)
        .map_err(|_| "AES init failed")?;
    let mut buf = ct_bytes.to_vec();
    let decrypted = cipher
        .decrypt_padded_vec_mut::<cbc::cipher::block_padding::Pkcs7>(&mut buf)
        .map_err(|_| "AES decryption failed")?;

    String::from_utf8(decrypted.to_vec())
        .map_err(|_| "decrypted content is not valid UTF-8")
}

// firmware/src/masters.rs
//
// Multi-master NVS storage. Each master occupies a numbered slot (0–7)
// with a secret, label, mode, cached pubkey, and connect secret.

use esp_idf_svc::nvs::{EspNvs, NvsDefault};
use heartwood_common::types::MasterMode;
use zeroize::Zeroize;

/// Maximum number of masters the device can hold.
pub const MAX_MASTERS: u8 = 8;

/// A loaded master identity (secret material in memory).
pub struct LoadedMaster {
    pub slot: u8,
    pub secret: [u8; 32],
    pub label: String,
    pub mode: MasterMode,
    pub pubkey: [u8; 32],
    pub connect_secret: [u8; 32],
}

impl Drop for LoadedMaster {
    fn drop(&mut self) {
        self.secret.zeroize();
        self.connect_secret.zeroize();
    }
}

/// Read the master count from NVS.
pub fn read_master_count(nvs: &EspNvs<NvsDefault>) -> u8 {
    let mut buf = [0u8; 1];
    match nvs.get_blob("master_count", &mut buf) {
        Ok(Some(b)) if b.len() == 1 => buf[0],
        _ => 0,
    }
}

/// Write the master count to NVS.
fn write_master_count(nvs: &mut EspNvs<NvsDefault>, count: u8) -> Result<(), &'static str> {
    nvs.set_blob("master_count", &[count])
        .map_err(|_| "failed to write master_count")
}

/// Load all masters from NVS into memory.
pub fn load_all(nvs: &EspNvs<NvsDefault>) -> Vec<LoadedMaster> {
    let count = read_master_count(nvs);
    let mut masters = Vec::with_capacity(count as usize);

    for slot in 0..count {
        match load_one(nvs, slot) {
            Some(m) => masters.push(m),
            None => log::warn!("Failed to load master slot {slot}"),
        }
    }

    masters
}

/// Load a single master from NVS.
fn load_one(nvs: &EspNvs<NvsDefault>, slot: u8) -> Option<LoadedMaster> {
    let prefix = format!("master_{slot}");

    let mut secret = [0u8; 32];
    let secret_key = format!("{prefix}_secret");
    match nvs.get_blob(&secret_key, &mut secret) {
        Ok(Some(b)) if b.len() == 32 => {}
        _ => return None,
    }

    let mut label_buf = [0u8; 32];
    let label_key = format!("{prefix}_label");
    let label = match nvs.get_blob(&label_key, &mut label_buf) {
        Ok(Some(b)) => String::from_utf8_lossy(b).to_string(),
        _ => "default".to_string(),
    };

    let mut mode_buf = [0u8; 1];
    let mode_key = format!("{prefix}_mode");
    let mode = match nvs.get_blob(&mode_key, &mut mode_buf) {
        Ok(Some(b)) if b.len() == 1 => MasterMode::from_u8(b[0]).unwrap_or(MasterMode::TreeMnemonic),
        _ => MasterMode::TreeMnemonic,
    };

    let mut pubkey = [0u8; 32];
    let pubkey_key = format!("{prefix}_pubkey");
    match nvs.get_blob(&pubkey_key, &mut pubkey) {
        Ok(Some(b)) if b.len() == 32 => {}
        _ => return None,
    }

    let mut connect_secret = [0u8; 32];
    let connect_key = format!("{prefix}_connect");
    match nvs.get_blob(&connect_key, &mut connect_secret) {
        Ok(Some(b)) if b.len() == 32 => {}
        _ => {
            // Generate a deterministic fallback if not present (shouldn't happen).
            log::warn!("No connect secret for slot {slot} — using zeroes");
        }
    }

    Some(LoadedMaster {
        slot,
        secret,
        label,
        mode,
        pubkey,
        connect_secret,
    })
}

/// Add a new master to NVS. Returns the assigned slot number.
pub fn add_master(
    nvs: &mut EspNvs<NvsDefault>,
    secret: &[u8; 32],
    label: &str,
    mode: MasterMode,
    pubkey: &[u8; 32],
    connect_secret: &[u8; 32],
) -> Result<u8, &'static str> {
    let count = read_master_count(nvs);
    if count >= MAX_MASTERS {
        return Err("maximum masters reached");
    }

    let slot = count;
    let prefix = format!("master_{slot}");

    nvs.set_blob(&format!("{prefix}_secret"), secret)
        .map_err(|_| "failed to write secret")?;
    nvs.set_blob(&format!("{prefix}_label"), label.as_bytes())
        .map_err(|_| "failed to write label")?;
    nvs.set_blob(&format!("{prefix}_mode"), &[mode as u8])
        .map_err(|_| "failed to write mode")?;
    nvs.set_blob(&format!("{prefix}_pubkey"), pubkey)
        .map_err(|_| "failed to write pubkey")?;
    nvs.set_blob(&format!("{prefix}_connect"), connect_secret)
        .map_err(|_| "failed to write connect secret")?;

    write_master_count(nvs, count + 1)?;

    log::info!("Added master slot {slot}: label={label}");
    Ok(slot)
}

/// Remove a master by slot index. Shifts higher slots down to fill the gap.
pub fn remove_master(
    nvs: &mut EspNvs<NvsDefault>,
    slot: u8,
) -> Result<(), &'static str> {
    let count = read_master_count(nvs);
    if slot >= count {
        return Err("slot out of range");
    }

    // Shift higher slots down.
    for i in slot..count - 1 {
        let src = format!("master_{}", i + 1);
        let dst = format!("master_{i}");

        // Copy each field from src to dst.
        for suffix in &["_secret", "_label", "_mode", "_pubkey", "_connect"] {
            let mut buf = [0u8; 64]; // large enough for any field
            let src_key = format!("{src}{suffix}");
            let dst_key = format!("{dst}{suffix}");
            if let Ok(Some(data)) = nvs.get_blob(&src_key, &mut buf) {
                let len = data.len();
                let _ = nvs.set_blob(&dst_key, &buf[..len]);
            }
        }
    }

    // Clear the last slot.
    let last = format!("master_{}", count - 1);
    for suffix in &["_secret", "_label", "_mode", "_pubkey", "_connect"] {
        let key = format!("{last}{suffix}");
        let _ = nvs.remove(&key);
    }

    write_master_count(nvs, count - 1)?;
    log::info!("Removed master slot {slot}");
    Ok(())
}

/// Find a master by x-only public key (32 bytes).
pub fn find_by_pubkey(masters: &[LoadedMaster], pubkey: &[u8; 32]) -> Option<usize> {
    masters.iter().position(|m| &m.pubkey == pubkey)
}

/// Handle a PROVISION_ADD frame (0x01). Returns the new LoadedMaster on success.
pub fn handle_add(
    usb: &mut UsbSerialDriver<'_>,
    frame: &Frame,
    nvs: &mut EspNvs<NvsDefault>,
    secp: &Arc<Secp256k1<secp256k1::SignOnly>>,
    display: &mut Display<'_>,
) -> Option<LoadedMaster> {
    // Legacy format: exactly 32 bytes = tree-mnemonic, label "default".
    let (mode, label, secret) = if frame.payload.len() == 32 {
        let mut secret = [0u8; 32];
        secret.copy_from_slice(&frame.payload);
        (MasterMode::TreeMnemonic, "default".to_string(), secret)
    } else if frame.payload.len() >= 2 + 32 {
        let mode_byte = frame.payload[0];
        let mode = match MasterMode::from_u8(mode_byte) {
            Some(m) => m,
            None => {
                log::warn!("Unknown provision mode: 0x{:02x}", mode_byte);
                protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
                return None;
            }
        };

        let label_len = frame.payload[1] as usize;
        if frame.payload.len() < 2 + label_len + 32 {
            log::warn!("Provision payload too short for label + secret");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return None;
        }

        let label = String::from_utf8_lossy(&frame.payload[2..2 + label_len]).to_string();
        let mut secret = [0u8; 32];
        secret.copy_from_slice(&frame.payload[2 + label_len..2 + label_len + 32]);

        (mode, label, secret)
    } else {
        log::warn!("Provision payload is {} bytes, expected >=32", frame.payload.len());
        protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        return None;
    };

    // Derive pubkey from secret.
    let keypair = match secp256k1::Keypair::from_seckey_slice(secp, &secret) {
        Ok(kp) => kp,
        Err(_) => {
            log::error!("Invalid secret key in provision payload");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return None;
        }
    };
    let (xonly, _) = keypair.x_only_public_key();
    let pubkey = xonly.serialize();

    // Generate a random connect secret (use the ESP32 hardware RNG).
    let mut connect_secret = [0u8; 32];
    esp_idf_svc::sys::esp_fill_random(connect_secret.as_mut_ptr() as *mut _, 32);

    // Store in NVS.
    match masters::add_master(nvs, &secret, &label, mode, &pubkey, &connect_secret) {
        Ok(slot) => {
            let npub = encode_npub(&pubkey);
            log::info!("Provisioned slot {slot}: {label} ({npub})");
            oled::show_npub(display, &npub);
            protocol::write_frame(usb, FRAME_TYPE_ACK, &[]);

            Some(LoadedMaster {
                slot,
                secret,
                label,
                mode,
                pubkey,
                connect_secret,
            })
        }
        Err(e) => {
            log::error!("Provision add failed: {e}");
            oled::show_error(display, "Provision failed");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            None
        }
    }
}

/// Handle a PROVISION_REMOVE frame (0x04).
pub fn handle_remove(
    usb: &mut UsbSerialDriver<'_>,
    frame: &Frame,
    nvs: &mut EspNvs<NvsDefault>,
    masters: &mut Vec<LoadedMaster>,
    display: &mut Display<'_>,
) {
    if frame.payload.len() != 1 {
        protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        return;
    }

    let slot = frame.payload[0];
    match masters::remove_master(nvs, slot) {
        Ok(()) => {
            // Remove from in-memory list.
            masters.retain(|m| m.slot != slot);
            // Re-number remaining masters to match NVS.
            for (i, m) in masters.iter_mut().enumerate() {
                m.slot = i as u8;
            }
            let msg = format!("Removed slot {slot}");
            oled::show_error(display, &msg);
            protocol::write_frame(usb, FRAME_TYPE_ACK, &[]);
        }
        Err(e) => {
            log::error!("Remove failed: {e}");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        }
    }
}

/// Handle a PROVISION_LIST frame (0x05). Responds with 0x07.
pub fn handle_list(
    usb: &mut UsbSerialDriver<'_>,
    masters: &[LoadedMaster],

        .iter()
        .map(|m| {

                "label": m.label,
                "mode": m.mode as u8,

        .collect();

    protocol::write_frame(usb, FRAME_TYPE_PROVISION_LIST_RESPONSE, json.as_bytes());
}

```rust
// firmware/src/policy.rs
//
// Client approval policy engine. Policies are held in RAM only —
// pushed from the bridge at session start and cleared on disconnect.

use std::time::Instant;

use heartwood_common::nip46::Nip46Method;
use heartwood_common::policy::{ApprovalTier, ClientPolicy};

/// Maximum concurrent client sessions.
pub const MAX_SESSIONS: usize = 32;

/// Rate limit: requests per window.
const RATE_LIMIT_MAX: u32 = 60;

/// Rate limit window in seconds.
const RATE_LIMIT_WINDOW_SECS: u64 = 60;

/// Per-master policy store.
pub struct MasterPolicies {
    pub master_slot: u8,
    pub policies: Vec<ClientPolicy>,
}

/// Per-client session state (rate limiting + active identity).
pub struct ClientSession {
    pub client_pubkey: [u8; 32],
    pub master_slot: u8,
    pub active_identity: Option<usize>,
    pub request_count: u32,
    pub window_start: Instant,
}

impl ClientSession {
    pub fn new(client_pubkey: [u8; 32], master_slot: u8) -> Self {
        Self {
            client_pubkey,
            master_slot,
            active_identity: None,
            request_count: 0,
            window_start: Instant::now(),
        }
    }

    /// Check and update rate limit. Returns true if within limit.
    pub fn check_rate_limit(&mut self) -> bool {
        let now = Instant::now();
        let elapsed = now.duration_since(self.window_start).as_secs();

        if elapsed >= RATE_LIMIT_WINDOW_SECS {
            // Reset window.
            self.request_count = 1;
            self.window_start = now;
            return true;
        }

        self.request_count += 1;
        self.request_count <= RATE_LIMIT_MAX
    }
}

/// The full policy state for the device.
pub struct PolicyEngine {
    /// Per-master policies (pushed from bridge).
    pub master_policies: Vec<MasterPolicies>,
    /// Active client sessions.
    pub sessions: Vec<ClientSession>,
    /// Whether a bridge session is authenticated.
    pub bridge_authenticated: bool,
}

impl PolicyEngine {
    pub fn new() -> Self {
        Self {
            master_policies: Vec::new(),
            sessions: Vec::new(),
            bridge_authenticated: false,
        }
    }

    /// Set policies for a master, replacing any existing ones.
    pub fn set_policies(&mut self, master_slot: u8, policies: Vec<ClientPolicy>) {
        self.master_policies.retain(|mp| mp.master_slot != master_slot);
        self.master_policies.push(MasterPolicies {
            master_slot,
            policies,
        });
    }

    /// Determine the approval tier for a request.
    pub fn check(
        &self,
        master_slot: u8,
        client_pubkey: &str,
        method: &Nip46Method,
        event_kind: Option<u64>,
    ) -> ApprovalTier {
        // If no bridge is authenticated, everything requires button.
        if !self.bridge_authenticated {
            return ApprovalTier::ButtonRequired;
        }

        // Methods that always auto-approve (ping, get_public_key, etc.).
        if method.always_auto_approve() {
            return ApprovalTier::AutoApprove;
        }

        // Methods that always require button (heartwood_derive, etc.).
        if method.always_requires_button() {
            return ApprovalTier::ButtonRequired;
        }

        // OLED-notify methods (heartwood_switch).
        if method.is_oled_notify() {
            return ApprovalTier::OledNotify;
        }

        // Look up client policy for this master.
        let policies = self
            .master_policies
            .iter()
            .find(|mp| mp.master_slot == master_slot);

        let policies = match policies {
            Some(mp) => &mp.policies,
            None => return ApprovalTier::ButtonRequired,
        };

        let policy = policies
            .iter()
            .find(|p| p.client_pubkey == client_pubkey);

        let policy = match policy {
            Some(p) => p,
            None => return ApprovalTier::ButtonRequired,
        };

        if !policy.auto_approve {
            return ApprovalTier::ButtonRequired;
        }

        // Check method is in allowed list.
        let method_str = method.as_str();
        if !policy.allowed_methods.is_empty()
            && !policy.allowed_methods.iter().any(|m| m == method_str)
        {
            return ApprovalTier::ButtonRequired;
        }

        // For sign_event, check kind is in allowed list.
        if matches!(method, Nip46Method::SignEvent) {
            if let Some(kind) = event_kind {
                if !policy.allowed_kinds.is_empty()
                    && !policy.allowed_kinds.contains(&kind)
                {
                    return ApprovalTier::ButtonRequired;
                }
            }
        }

        ApprovalTier::AutoApprove
    }

    /// Find or create a client session. Returns mutable reference.
    pub fn get_or_create_session(
        &mut self,
        client_pubkey: [u8; 32],
        master_slot: u8,
    ) -> Option<&mut ClientSession> {
        // Find existing.
        let existing = self.sessions.iter().position(|s| {
            s.client_pubkey == client_pubkey && s.master_slot == master_slot
        });

        if let Some(idx) = existing {
            return Some(&mut self.sessions[idx]);
        }

        // Create new if under limit.
        if self.sessions.len() >= MAX_SESSIONS {
            log::warn!("Max sessions reached — rejecting new client");
            return None;
        }

        self.sessions.push(ClientSession::new(client_pubkey, master_slot));
        self.sessions.last_mut()
    }

    /// Clear all state (bridge disconnected).
    pub fn clear(&mut self) {
        self.master_policies.clear();
        self.sessions.clear();
        self.bridge_authenticated = false;
    }
}

// Bridge session management. The bridge authenticates with a shared secret
// (stored in NVS at provision time), then pushes client policies.

use esp_idf_hal::usb_serial::UsbSerialDriver;
use esp_idf_svc::nvs::{EspNvs, NvsDefault};

use heartwood_common::frame::Frame;
use heartwood_common::policy::ClientPolicy;
use heartwood_common::types::*;

use crate::policy::PolicyEngine;
use crate::protocol;

const NVS_BRIDGE_SECRET_KEY: &str = "bridge_secret";

/// Read the bridge authentication secret from NVS.
pub fn read_bridge_secret(nvs: &EspNvs<NvsDefault>) -> Option<[u8; 32]> {
    let mut buf = [0u8; 32];
    match nvs.get_blob(NVS_BRIDGE_SECRET_KEY, &mut buf) {
        Ok(Some(b)) if b.len() == 32 => Some(buf),
        _ => None,
    }
}

/// Write the bridge authentication secret to NVS.
pub fn write_bridge_secret(
    nvs: &mut EspNvs<NvsDefault>,
    secret: &[u8; 32],
) -> Result<(), &'static str> {
    nvs.set_blob(NVS_BRIDGE_SECRET_KEY, secret)
        .map_err(|_| "failed to write bridge secret")
}

/// Handle a SESSION_AUTH frame (0x21).
pub fn handle_auth(
    usb: &mut UsbSerialDriver<'_>,
    frame: &Frame,
    nvs: &EspNvs<NvsDefault>,
    policy_engine: &mut PolicyEngine,
) {
    if frame.payload.len() != 32 {
        log::warn!("SESSION_AUTH payload is {} bytes, expected 32", frame.payload.len());
        protocol::write_frame(usb, FRAME_TYPE_SESSION_ACK, &[0x01]); // 0x01 = failed
        return;
    }

    let bridge_secret = match read_bridge_secret(nvs) {
        Some(s) => s,
        None => {
            log::warn!("No bridge secret in NVS — cannot authenticate");
            protocol::write_frame(usb, FRAME_TYPE_SESSION_ACK, &[0x02]); // 0x02 = no secret configured
            return;
        }
    };

    // Constant-time comparison.
    let mut diff = 0u8;
    for (a, b) in frame.payload.iter().zip(bridge_secret.iter()) {
        diff |= a ^ b;
    }

    if diff != 0 {
        log::warn!("Bridge authentication failed — wrong secret");
        policy_engine.bridge_authenticated = false;
        protocol::write_frame(usb, FRAME_TYPE_SESSION_ACK, &[0x01]); // 0x01 = failed
        return;
    }

    log::info!("Bridge authenticated successfully");
    policy_engine.bridge_authenticated = true;
    protocol::write_frame(usb, FRAME_TYPE_SESSION_ACK, &[0x00]); // 0x00 = success
}

/// Handle a POLICY_PUSH frame (0x20).
pub fn handle_policy_push(
    usb: &mut UsbSerialDriver<'_>,
    frame: &Frame,
    masters: &[LoadedMaster],
    policy_engine: &mut PolicyEngine,
) {
    if !policy_engine.bridge_authenticated {
        log::warn!("Policy push rejected — bridge not authenticated");
        protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        return;
    }

    if frame.payload.len() < 32 {
        log::warn!("Policy push payload too short");
        protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        return;
    }

    let master_pubkey: [u8; 32] = frame.payload[..32].try_into().unwrap();
    let policy_json = &frame.payload[32..];

    // Find the master.
    let master_idx = match crate::masters::find_by_pubkey(masters, &master_pubkey) {
        Some(idx) => idx,
        None => {
            log::warn!("Policy push for unknown master pubkey");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };

    // Parse policies.
    let policies: Vec<ClientPolicy> = match serde_json::from_slice(policy_json) {
        Ok(p) => p,
        Err(e) => {
            log::warn!("Failed to parse policies: {e}");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };

    let slot = masters[master_idx].slot;
    let count = policies.len();
    policy_engine.set_policies(slot, policies);
    log::info!("Loaded {count} client policies for master slot {slot}");
    protocol::write_frame(usb, FRAME_TYPE_ACK, &[]);
}

// Boot sequence:
//   1. Initialise peripherals (LED, Vext, OLED, button, USB serial, NVS)
//   2. Load all masters from NVS

use secp256k1::Secp256k1;

    esp_idf_svc::sys::link_patches();
    esp_idf_svc::log::EspLogger::initialize_default();

    log::info!("Heartwood ESP32 — Phase 4 (full NIP-46 HSM)");

    // Turn on white LED (GPIO 35, active high).
    let mut led = PinDriver::output(peripherals.pins.gpio35).expect("LED pin");

        peripherals.pins.gpio19,

                protocol::write_frame(&mut usb, FRAME_TYPE_NACK, &[]);

                    provision::handle_add(&mut usb, &frame, &mut nvs, &secp, &mut display)
                {
                    loaded_masters.push(master);
                }
            }
            FRAME_TYPE_PROVISION_REMOVE => {
                provision::handle_remove(
                    &mut usb,
                    &frame,
                    &mut nvs,
                    &mut loaded_masters,
                    &mut display,

            FRAME_TYPE_PROVISION_LIST => {
                provision::handle_list(&mut usb, &loaded_masters);

                        &mut usb,
                        &frame,
                        &master.secret,
                        &master.label,
                        master.mode,
                        master.slot,
                        &secp,
                        &mut display,
                        &button_pin,
                        &mut policy_engine,
                    );
                    oled::show_error(&mut display, "Heartwood ready");
                } else {
                    protocol::write_frame(&mut usb, FRAME_TYPE_NACK, &[]);
                }
            }

            // Bridge session management.
            FRAME_TYPE_SESSION_AUTH => {
                session::handle_auth(&mut usb, &frame, &nvs, &mut policy_engine);
                if policy_engine.bridge_authenticated {
                    oled::show_bridge_connected(
                        &mut display,
                        loaded_masters.len() as u8,
                        0,
                    );
                }
            }
            FRAME_TYPE_POLICY_PUSH => {
                session::handle_policy_push(
                    &mut usb,
                    &frame,
                    &loaded_masters,
                    &mut policy_engine,
                );
            }

            // Encrypted NIP-46 (bridge mode).
            FRAME_TYPE_ENCRYPTED_REQUEST => {
                if !policy_engine.bridge_authenticated {
                    log::warn!("Encrypted request rejected — bridge not authenticated");
                    protocol::write_frame(&mut usb, FRAME_TYPE_NACK, &[]);

                    // TODO: Task 12 will implement transport decryption here.
                    // For now, NACK encrypted requests.
                    log::warn!("Encrypted transport not yet implemented");
                    protocol::write_frame(&mut usb, FRAME_TYPE_NACK, &[]);
                }

            _ => {
                log::warn!("Unknown frame type: 0x{:02x}", frame.frame_type);
                protocol::write_frame(&mut usb, FRAME_TYPE_NACK, &[]);
            }
        }
    }
}

// firmware/src/transport.rs
//
// On-device NIP-44 transport encryption/decryption.
//
// Decrypts inbound 0x10 frames, routes to NIP-46 handler, encrypts
// the response, and returns a 0x11 frame.

use esp_idf_hal::gpio::{Input, PinDriver};
use esp_idf_hal::usb_serial::UsbSerialDriver;
use secp256k1::{Secp256k1, SignOnly};
use std::sync::Arc;

use heartwood_common::frame::Frame;
use heartwood_common::nip44;
use heartwood_common::types::*;

use crate::masters::LoadedMaster;
use crate::oled::Display;
use crate::policy::PolicyEngine;
use crate::protocol;

/// Handle an encrypted NIP-46 request frame (0x10).
///
/// Payload: [master_pubkey_32][client_pubkey_32][ciphertext...]
pub fn handle_encrypted_request(
    usb: &mut UsbSerialDriver<'_>,
    frame: &Frame,
    masters: &[LoadedMaster],
    secp: &Arc<Secp256k1<SignOnly>>,
    display: &mut Display<'_>,
    button_pin: &PinDriver<'_, Input>,
    policy_engine: &mut PolicyEngine,
) {
    if frame.payload.len() < 64 {
        log::warn!("Encrypted request payload too short ({} bytes)", frame.payload.len());
        protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
        return;
    }

    let master_pubkey: [u8; 32] = frame.payload[..32].try_into().unwrap();
    let client_pubkey: [u8; 32] = frame.payload[32..64].try_into().unwrap();
    let ciphertext_bytes = &frame.payload[64..];

    // Find the master.
    let master_idx = match crate::masters::find_by_pubkey(masters, &master_pubkey) {
        Some(idx) => idx,
        None => {
            log::warn!("Encrypted request for unknown master pubkey");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };
    let master = &masters[master_idx];

    // Derive conversation key.
    let conversation_key = match nip44::get_conversation_key(&master.secret, &client_pubkey) {
        Ok(ck) => ck,
        Err(e) => {
            log::error!("Conversation key derivation failed: {e}");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };

    // Decrypt the ciphertext (base64-encoded NIP-44).
    let ciphertext_b64 = match std::str::from_utf8(ciphertext_bytes) {
        Ok(s) => s,
        Err(_) => {
            log::error!("Ciphertext is not valid UTF-8");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };

    let plaintext_json = match nip44::decrypt(&conversation_key, ciphertext_b64) {
        Ok(pt) => pt,
        Err(e) => {
            log::error!("NIP-44 decrypt failed: {e}");
            protocol::write_frame(usb, FRAME_TYPE_NACK, &[]);
            return;
        }
    };

    log::info!("Decrypted NIP-46 request ({} bytes)", plaintext_json.len());

    // Build a plaintext frame for the NIP-46 handler.
    let inner_frame = Frame {
        frame_type: FRAME_TYPE_NIP46_REQUEST,
        payload: plaintext_json.as_bytes().to_vec(),
    };

    // Capture the response by temporarily redirecting output.
    // We need the handler to produce a response JSON string.
    // For now, call the handler and capture the response frame it would send.
    //
    // TODO: Refactor nip46_handler to return the response JSON instead of
    // writing directly to USB. For now, we process inline.
    crate::nip46_handler::handle_request(
        usb, // This will write the response as a plaintext 0x03 frame — we need to intercept.
        &inner_frame,
        &master.secret,
        &master.label,
        master.mode,
        master.slot,
        secp,
        display,
        button_pin,
        policy_engine,
    );

    // NOTE: The current architecture has the handler write directly to USB.
    // To encrypt the response, we need to refactor the handler to return
    // the response JSON. This is a known limitation that will be addressed
    // when we implement the full encrypted transport flow.
    //
    // For the initial integration, the handler writes a plaintext 0x03 frame.
    // The bridge must handle both 0x03 (plaintext) and 0x11 (encrypted) for
    // backward compatibility during the transition.
}
```

/// Send an encrypted NIP-46 request to the ESP32 via frame type 0x10.
///
/// Payload: [master_pubkey_32][client_pubkey_32][ciphertext_bytes...]
fn forward_encrypted(
    port: &mut Box<dyn serialport::SerialPort>,
    master_pubkey: &[u8; 32],
    client_pubkey: &[u8; 32],
    ciphertext: &str,
) -> Result<String, String> {
    let mut payload = Vec::with_capacity(64 + ciphertext.len());
    payload.extend_from_slice(master_pubkey);
    payload.extend_from_slice(client_pubkey);
    payload.extend_from_slice(ciphertext.as_bytes());

    let frame_bytes = frame::build_frame(FRAME_TYPE_ENCRYPTED_REQUEST, &payload)
        .map_err(|e| format!("frame build failed: {:?}", e))?;

    std::io::Write::write_all(port.as_mut(), &frame_bytes)
        .map_err(|e| format!("serial write failed: {e}"))?;
    std::io::Write::flush(port.as_mut())
        .map_err(|e| format!("serial flush failed: {e}"))?;

    // Read response — accept both 0x03 (plaintext, transition) and 0x11 (encrypted).
    read_response(port)
}

/// Authenticate the bridge session with the ESP32.
fn authenticate_bridge(
    port: &mut Box<dyn serialport::SerialPort>,
    bridge_secret: &[u8; 32],
) -> Result<(), String> {
    let frame_bytes = frame::build_frame(FRAME_TYPE_SESSION_AUTH, bridge_secret)
        .map_err(|e| format!("frame build failed: {:?}", e))?;

    std::io::Write::write_all(port.as_mut(), &frame_bytes)
        .map_err(|e| format!("serial write failed: {e}"))?;
    std::io::Write::flush(port.as_mut())
        .map_err(|e| format!("serial flush failed: {e}"))?;

    // Read SESSION_ACK (0x22).
    let mut buf = vec![0u8; MAX_PAYLOAD_SIZE + FRAME_OVERHEAD];
    let mut pos = 0;
    let deadline = std::time::Instant::now() + Duration::from_secs(10);

    loop {
        if std::time::Instant::now() > deadline {
            return Err("timeout waiting for session ACK".into());
        }

        let mut byte = [0u8; 1];
        match port.read(&mut byte) {
            Ok(1) => {
                if pos < buf.len() {
                    buf[pos] = byte[0];
                    pos += 1;
                }

                if pos >= FRAME_OVERHEAD {
                    match frame::parse_frame(&buf[..pos]) {
                        Ok(f) if f.frame_type == FRAME_TYPE_SESSION_ACK => {
                            if f.payload.first() == Some(&0x00) {
                                log::info!("Bridge session authenticated");
                                return Ok(());
                            } else {
                                return Err("bridge authentication failed".into());
                            }
                        }
                        Err(frame::FrameError::TooShort) => continue,
                        _ => continue,
                    }
                }
            }
            Ok(_) => {}
            Err(ref e) if e.kind() == std::io::ErrorKind::TimedOut => {}
            Err(e) => return Err(format!("serial read error: {e}")),
        }
    }
}
